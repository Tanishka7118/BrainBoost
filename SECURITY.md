# Security Policy

## Supported Versions

BrainBoost is currently maintained on the latest `main` branch.
Security fixes are applied to `main` and included in the next release.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately by emailing:

- maintainers@brainboost.dev

Include the following details:

- vulnerability type and impact
- exact reproduction steps
- affected endpoints/files
- proof-of-concept or logs
- suggested remediation if available

## Response Expectations

- Initial response: within 72 hours
- Triage decision: within 7 days
- Fix timeline: based on severity and complexity

We appreciate responsible disclosure and will coordinate remediation before any
public disclosure.
